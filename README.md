# LinguaKids monorepo

Trilingual kids learning app (English · Arabic · Urdu).

## Structure

```
apps/mobile/     React Native app (Android + iOS)
backend/         API server (Node.js + PostgreSQL)
cms/             Content management
assets/          Shared media (fonts, audio, images)
```

## Quick start (Android)

```powershell
cd apps/mobile
npm install
npm run android
```

See [TECH_STACK.md](./TECH_STACK.md) for versions and Windows build notes.
